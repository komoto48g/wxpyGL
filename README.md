# wxpyGL

Python package based on OpenGL/GLUT
with wxPython shell extension library


## Getting Started

### Prerequisites
- Python 3.5 or later
    - numpy
    - pillow
    - matplotlib
    - wxpython >= 4.1.1
    - mwxlib

### How to use
```
py wxmatgl.py
```
